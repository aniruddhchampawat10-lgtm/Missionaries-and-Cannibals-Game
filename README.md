## 🎮 Missionaries and Cannibals Game – Description

The **Missionaries and Cannibals Game** is a classic problem-based puzzle implemented using the **Pygame** library in Python. The objective of the game is to safely transport **three missionaries and three cannibals** from one side of a river to the other using a boat with a maximum capacity of two people.

### 🧠 Game Logic

The challenge lies in following a strict rule:

* At no point on either side of the river should the number of cannibals exceed the number of missionaries.
* If this condition is violated, the cannibals will "eat" the missionaries, resulting in a **game over**.

### 🚤 Gameplay Mechanics

* The player selects who boards the boat (1 or 2 people).
* The boat moves between the two shores.
* Each move must maintain a safe balance between missionaries and cannibals.
* The game continues until either:

  * All characters safely reach the other side (**You Win 🎉**), or
  * An unsafe condition occurs (**Game Over ❌**).

### 🎨 Features

* Interactive graphical interface using Pygame
* Clear visual representation of both shores and characters
* Real-time updates based on player actions
* Game states: New Game, Game Over, and Game Won

### ▶️ How to Run

1. Install Pygame:

   ```bash
   pip install pygame
   ```
2. Run the game:

   ```bash
   python main.py
   ```
